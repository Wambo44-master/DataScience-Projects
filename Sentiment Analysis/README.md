# To Vaccinate or Not To Vaccinate - Sentiment Analysis

## 🎯 Problem
The objective of this challenge is to develop a machine learning model to assess if a Twitter post related to vaccinations is positive, neutral, or negative.

## 📊 Dataset
- Source: Zindi
- Size: 10001 rows, 4 columns
- Target:  Apply supervised learning techniques to solve a real-world problem using data from Zindi Africa

## 🔧 My Approach
1. Data cleaning
2. Feature engineering  
3. Data visualization techniques
4. Supervised Learning
5.  Text vectorization
6. Natural Language processing

 
## 🧠 What I Learned
- Data visualization increases comprehension and understanding of data
- Missing values either need to be dropped or filled using 0, mean or median depending on the type of data
