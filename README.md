# Zindi Africa Data Science Projects

## 👋 About
This repository contains my submissions for Zindi data science challenges completed during my data science training at Ngao Labs.

## 🎯 Why Zindi?
Zindi is Africa's premier data science competition platform. These challenges use real African datasets to solve real problems in healthcare, finance, agriculture, and public policy.
