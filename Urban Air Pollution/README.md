# Urban Air Pollution

## 🎯 Problem
Predict PM2.5 concentration levels for different cities using unsupervised learning techniques.

## 📊 Dataset
- Size: 30557 rows, 82 columns
- Target:  Provide a unified and world-wide air quality information.

## 🔧 My Approach
1. Data cleaning
2. Feature engineering  
3. Data visualization techniques
4. Unsupervised learning

## 🧠 What I Learned
- Data visualization increases comprehension and understanding of data
- Missing values either need to be dropped or filled using 0, mean or median depending on the type of data
