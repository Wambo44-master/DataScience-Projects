# Financial Inclusion in Africa

## 🎯 Problem 
Access to bank account has traditionally been used as an indicator of financial inclusion. The objective of this project is to create a machine learning model to predict which individuals are most likely to have or use a bank account. The models and solutions developed can provide an indication of the state of financial inclusion in Kenya, Rwanda, Tanzania and Rwanda.

- Size: [23524 rows, 13 columns]
- Target: Predict which individuals are most likely to have or use a bank account.

## 🔧 My Approach
1. Data cleaning
2. Feature engineering  
3. Model: Logistic Regression, Random Forest Classifier, XGB Boost

## 🧠 What I Learned
- Feature Engineering creates new insights for data.
- XGB Boost performed best
